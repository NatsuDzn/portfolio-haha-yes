## 🏁 Setup <a name = "getting_started"></a>

### Prerequisites

You need

```
node-js (npm)
```

### Installing

Clone and cd into the repository and run `npm install`

### Development mode

`npm run dev`

Open `localhost:1234`

### Production mode

`npm run build`

Serve the `dist` folder.
